核心概念

三個關鍵步驟

工作整合也好，電影訂票也好，背後的邏輯相同：你授權、Claude 協調、資訊整合回傳給你。

Step 01
🔐

授權連結

你在 Claude 介面授權，Claude 取得各平台的 OAuth Token 或 API Key。你決定授權哪些服務、哪些資料範圍，以及允許哪些動作。

Step 02
🌊

資料擷取

根據你的自然語言指令，Claude 決定需要查詢哪些平台，在已授權範圍內擷取資料——可以是工作文件，也可以是電影評分。

Step 03
📊

整合輸出

Claude 整合多來源資訊，依你的格式偏好輸出結果——可以是工作摘要，也可以是附上真實訂票連結的電影推薦。
